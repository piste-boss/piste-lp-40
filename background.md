# プロジェクト背景と仮説：40-50代向け新LP制作

## 1. 背景
- **実施内容**: 2026年1月、既存のLP（[Piste LP](https://piste-lp.netlify.app/)）を用いてインスタ広告を配信。
- **配信結果**: 
    - 40-50代を中心にリンククリック数が多く発生した。
    - しかし、既存LPのターゲット層（産後ダイエット・託児希望のママ層）と実際のクリック層がアンマッチであったため、その多くがコンバージョンに至らず離脱した。

## 2. 課題
- 広告に対する反応（興味）はあるものの、着地先であるLPの内容がターゲット（40-50代）のニーズに合致していない。

## 3. 仮説
- **需要の存在**: 40-50代におけるパーソナルトレーニングの潜在的需要は非常に高い。
- **解決策**: この年齢層に特化した（悩みやベネフィットを最適化した）LPを制作・運用することで、高い成約率で新規顧客を取り込むことができる。

## 4. 今後のアクション
- 40-50代をメインターゲットとしたLPを新規に作成する。
- 既存の「産後・託児」とは異なる、この層特有の悩み（健康維持、更年期対策、体型変化、自己投資など）に響くメッセージ・デザインを検討する。
