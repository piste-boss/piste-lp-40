# 【決定版】Piste LP 画像・ビジュアル指示書

**ターゲット**：40代〜50代（男女）  
**コンセプト**：鍛える前に「整える」。大人のためのメンテナンス・ジム。

---

## ■ 鉄の掟（General Rules）

### 日本人モデル限定
- **絶対NG**：外国人モデル。どんなにカッコよくても使用禁止。
- **理由**：「自分事」として捉えてもらうため。また、怪しい広告に見せないため。
- **理想**：実際の会員様、または「碧南にいそうな親しみやすい雰囲気」の日本人モデル素材。

### 「汗」よりも「笑顔」
- 歯を食いしばる表情や、大量の汗はNG。
- 「爽やか」「スッキリ」「納得」といったポジティブで穏やかな表情を採用する。

### 清潔感の徹底
- 背景に余計なもの（乱雑なタオル、汚れ）が写り込まないようにする。

---

## 1. ファーストビュー（FV）
- **【目的】** サイトを開いた瞬間に「ここなら安心だ（自分に合っている）」と思わせる。
- **配置**：メインビジュアル（背景または右側配置）
- **被写体**：
    - 40代〜50代の日本人（男女どちらでも可）。
    - 石川代表（トレーナー）とのツーショット推奨。
- **シーン**：
    - 笑顔で会話をしているカウンセリング風景。
    - または、軽い負荷でフォーム確認をしている最中の、穏やかなトレーニング風景。
- **トーン**：自然光が入るような、明るく清潔感のある雰囲気。

## 2. 導入・共感（Intro）
- **【目的】** ターゲットの悩み（痛み・疲れ）を視覚化し、「そうそう！」と思わせる。
- **配置**：「こんなお悩みありませんか？」エリア
- **被写体**：40-50代の日本人ビジネスパーソンや主婦。
- **シーン**：
    - デスクワーク中に肩や腰をトントンと叩いている様子。
    - 「ふぅ」とため息をついている、日常の疲れを感じさせるワンシーン。
- **トーン**：FVよりも少し彩度を落とし、**「悩み＝暗い・重い」**演出をする。

## 3. 解決策（Solution）「整える」
- **【目的】** Piste最大の強みである「柔軟性・ストレッチ」を証明する。
- **配置**：「不調に寄り添うストレッチ提案」セクション
- **必須シーン**：
    - トレーナーが会員に「パートナーストレッチ」を行っている写真。（施術ベッドやマット上で、足を伸ばしてあげている等）
- **表情**：会員様が**「あぁ、そこそこ…（気持ちいい）」**とリラックスして目を閉じている、または和らいだ表情。
- **狙い**：「運動しなきゃ」というプレッシャーを消し、「癒やされに行く」動機を作る。

## 4. 信頼の証拠（Reasons）
- **【目的】** 言葉の裏付け（エビデンス）を見せる。
- **医療連携エリア**：
    - 「医療法人つばさクリニック」の外観写真（可能なら）。
    - または、清潔感のある「聴診器とカルテ」などの医療イメージ画像。
- **科学的アプローチエリア**：
    - VBTデバイス（速度計測器）やタブレット端末の画面を見せている手元のアップ。「感覚ではなく数値」であることを直感的に伝える。
- **店舗環境エリア**：
    - 店内の広角写真。整理整頓された器具、清潔な床。圧迫感のないプライベート空間であることを伝える。

## 5. 価格比較アンカリング（Anchor / Graph）
- **【目的】** 視覚的なインパクトで「圧倒的なお得感」を植え付ける。
- **形式**：イラストまたは図解による棒グラフ
- **構成要素**：
    - **左のバー（一般的ジム）**：色：グレー（無機質で高い印象） / 高さ：100% / ラベル：「一般的なパーソナルジム」
    - **右のバー（Piste）**：色：ゴールド（金属光沢グラデーション） / 高さ：左の約1/3 / ラベル：「Piste（ピステ）」
- **必須演出**：
    - 左のバーの頂点から右のバーの頂点へ、**下向きの「点線矢印」**を入れる。
    - その点線付近に**「コストを大幅カット」「約1/3の価格」**と記載。
    - ゴールドのバーの上に王冠アイコン等を乗せ、「正解」であることを強調。

## 6. トレーナー紹介（Profile）
- **【目的】** 「この人なら怖くない、話を聞いてくれそう」という親近感を作る。
- **被写体**：石川代表（バストアップ〜腰上のショット）。
- **ポーズ・表情**：
    - 腕組み（威圧感）はNG。手は体の前で軽く組むか自然に下ろす。
    - 優しく微笑む、穏やかな表情。「先生」ではなく「パートナー」としての顔つき。
- **服装**：清潔感のあるPisteのユニフォーム（ポロシャツ等）。

## 7. クロージング（CTA）
- **【目的】** 読後感を良くし、申し込みボタンを押させる。
- **配置**：申し込みボタン周辺の背景画像
- **シーン**：
    - セッションを終えて、スッキリした顔をしている会員様（日本人）。
    - または、トレーナーと会員様が「お疲れ様でした！」と談笑しているシーン。
- **トーン**：最も明るく、ポジティブな色味。「未来の自分」を想像させる。

---

## 制作担当者への伝言メモ
「カッコよさ」よりも「リアリティ」を最優先してください。デザイン的に少し泥臭くなっても構いません。「地元の40-50代が、気負わずに通えそうな雰囲気」を作ることが最重要ミッションです。おしゃれすぎる外国人素材や、高級すぎるホテルライクな演出は避けてください。
