# Piste ランディングページ分析結果

## 1. サービス概要
- **名称**: Piste（ピステ）
- **コンセプト**: 子連れで通える、ママのためのパーソナルトレーニングジム
- **場所**: 愛知県碧南市
- **強み**: 
    - 託児無料（保育士常駐）
    - 1回20分の超効率トレーニング
    - 産後ダイエットに特化

## 2. 料金プラン
| プラン名 | 料金 (月額) | 内容 | 特徴 |
| :--- | :--- | :--- | :--- |
| **プラチナ** | 20,000円 | 月8回 | 24時間ジム利用可、食事指導付 |
| **スタンダード** | 13,000円 | 月4回 | 24時間ジム利用可、食事指導付（一番人気） |
| **ライト** | 9,980円 | 月2回 | ジム利用時間制限あり |

## 3. 現在のキャンペーン
- **入会特典**: 入会金（11,000円）＋ 初月月謝 ＝ **0円**
- **条件**: 毎月先着3名様限定
- **体験**: 無料体験レッスン受付中

## 4. ページ構成
1. **ヒーローセクション**: 「結果が出る、託児無料」のキャッチコピー
2. **悩み共感**: 「産後太り」「時間がない」などのママの課題
3. **Pisteが選ばれる理由**: 専門の保育、高効率トレーニング、通いやすさ
4. **トレーナー紹介**: 代表・石川峻（パパ・元エンジニア）
5. **お客様の声**: 実際のビフォーアフターや体験談
6. **利用の流れ**: 予約 → カウンセリング・体験 → 開始
7. **Q&A / アクセス**: よくある質問、地図、営業時間

## 5. デザイン・テクニカル評価
- **カラー**: 白、パステルグリーン、オレンジ（清潔感と安心感）
- **ビジュアル**: 実際の託児風景やトレーニング中の笑顔の写真を多用
- **UX**: モバイル対応済み、画面下部に常駐するコンバージョンボタン（無料体験予約）が設置
- **エラーチェック**: リンク切れ、画像表示エラー等は確認されませんでした。
